# About
Aros' fletching is about creating arrows with new different mechanics.
Mod adds a few new arrows, including:
- Beginner arrow
- Crimson arrow
- Warped arrow
- Glowing arrow
- Messenger arrow
- Prismarine arrow
- Tnt arrow
- Terracotta arrow
- "Sight"
- "Oblivion"

A few effects:
- Rage
- Confusion

And enchanted feather with whistle mechanic
